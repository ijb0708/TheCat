# TheCat
키보드로 마우스를 제어하기위해 만든프로그램입니다.

마우스 제어 프로그램
목적: 코딩시 마우스 사용 최대한 배제
기능:
- 화살표 및 hjkl  로 이동가능 (이동이 매우 매끄러울 것!)
- 위치 를 숫자칸에 예약가능 (추후 클릭나 매크로도 염두)
- 사용자 설정 가능

마우스 좌표
마우스이동 제어기
매크로 및 위치 저장 및 불러오기
가용 쓰래드(쓰래드를 껏타 킬것인가 아님 단일로 할것인가?)
키맵핑

자바x

백그라운드 실행시 키입력 받을 수 있어야 할것!
